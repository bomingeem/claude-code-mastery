# 개발자 웹 이력서 개발 로드맵

## 프로젝트 개요
HTML, CSS, JavaScript, Tailwind CSS를 사용하여 전문적인 개발자 웹 이력서 구축

---

## Phase 1: 프로젝트 설정 및 계획
- [ ] 프로젝트 폴더 구조 생성
  - `index.html` - 메인 페이지
  - `styles.css` - 커스텀 스타일 (필요시)
  - `script.js` - 인터랙티브 기능
  - `assets/` - 이미지, 아이콘 폴더
- [ ] Tailwind CSS CDN 또는 설치 설정
- [ ] Git 저장소 초기화
